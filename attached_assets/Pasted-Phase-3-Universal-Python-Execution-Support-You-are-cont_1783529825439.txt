Project Overview

This project is a Python Code Execution Visualizer.

Goal:

A user pastes any Python code into the web editor.

The backend executes the code using sys.settrace.

Every executed line generates a snapshot.

The frontend plays those snapshots as an interactive visualization.

The final experience should feel similar to Python Tutor, but with a modern React UI and animated data-structure visualizations.
