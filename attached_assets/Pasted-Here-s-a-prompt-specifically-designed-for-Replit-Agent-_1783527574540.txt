# Project Overview

This project is a **Python Execution Visualizer** similar to Python Tutor.

Users submit Python code through a React/Vite frontend.

The backend is written in Flask.

The backend safely executes Python code while tracing execution.

Execution is converted into immutable snapshots.

Snapshots are returned through:

* POST /api/run

The frontend visualizes those snapshots step-by-step.

The backend uses Python's tracing system (sys.settrace) to record execution.

The project has already gone through multiple implementation phases.
